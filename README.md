# Cloud Cost Estimator

## Project Overview

Cloud Cost Estimator is a Flask-based web application designed to estimate the monthly cost of cloud infrastructure resources. The application allows users to calculate the estimated cost of AWS services such as EC2, S3, RDS, and Data Transfer based on user input.

The project provides a simple dashboard, user registration system, cost estimation engine, cost history tracking, reports, and an admin dashboard.

---

## Features

### User Features

* User Registration
* User Login
* AWS Cost Estimation
* EC2 Cost Calculation
* S3 Storage Cost Calculation
* RDS Cost Calculation
* Data Transfer Cost Calculation
* Cost History Tracking
* Cost Analytics Dashboard
* Cost Optimization Suggestions

### Admin Features

* View Total Users
* View Total Estimates
* Monitor Recent Cost Estimates
